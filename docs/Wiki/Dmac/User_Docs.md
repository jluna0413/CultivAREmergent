# CultivAR User Documentation

## Collection Page

The Collection page allows you to manage your cannabis cultivar database. You can view, add, edit, and delete cultivars from this page.

### Adding a Cultivar

To add a new cultivar to your collection:

1. Click the "Add Cultivar" button at the top right of the page.
2. Fill in the cultivar details in the form:
   - Name (required): The name of the cultivar
   - Breeder: Select a breeder from the dropdown or add a new one
   - Genetics: Adjust the Indica/Sativa percentages using the sliders
   - Autoflower: Check this box if the cultivar is an autoflowering variety
   - Cycle Time: The average number of days from seed to harvest
   - Seed Count: The number of seeds you have in stock
   - URL: A link to more information about the cultivar
   - Description: A detailed description of the cultivar
3. Click "Save Cultivar" to add the cultivar to your collection.

### Adding a Breeder

If the breeder of your cultivar is not in the dropdown list, you can add a new one:

1. Click "Add new breeder" below the breeder dropdown.
2. Enter the name of the breeder in the form.
3. Click "Save Breeder" to add the breeder to the database.
4. The new breeder will be automatically selected in the cultivar form.

### Editing a Cultivar

To edit an existing cultivar:

1. Find the cultivar you want to edit in the list.
2. Click the "Edit" button (pencil icon) on the cultivar card.
3. Update the cultivar details in the form.
4. Click "Update Cultivar" to save your changes.

### Deleting a Cultivar

To delete a cultivar from your collection:

1. Find the cultivar you want to delete in the list.
2. Click the "Delete" button (trash icon) on the cultivar card.
3. Confirm the deletion when prompted.

**Note:** You cannot delete a cultivar that is currently being used by any plants in your garden.

### Filtering and Searching

You can filter and search your cultivar collection:

1. Use the "Type" dropdown to filter by Indica, Sativa, or Hybrid.
2. Use the "Breeder" dropdown to filter by a specific breeder.
3. Use the "Autoflower" dropdown to filter by autoflowering or photoperiod cultivars.
4. Use the search box to find cultivars by name.

### Viewing Modes

You can view your cultivar collection in two different modes:

1. Grid View: Displays cultivars as cards in a grid layout (default).
2. List View: Displays cultivars in a compact list format.

Click the grid or list icons in the top left to switch between views.
